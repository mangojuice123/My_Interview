# 常见问题
## Redis 支持哪几种数据类型？
答：
- Redis 数据库里面的每个 key-value pair 都是由对象组成的。
- 数据库的键总是一个 String object。
- 数据库的值可以是五种对象中的一种：
  - String：做一些复杂计数功能的缓存；
  - List：实现简单的消息队列；
  - Hash：实现单点登陆功能，cookieId 作为 key，设置 30 分钟为缓存过期时间，模拟出 session 的效果；
  - Set：利用交集、并集、差集等功能，计算共同喜好等；
  - Sorted Set：做排行榜应用或者进行范围查找。
