# 题解
```
 public ListNode reverseKGroup(ListNode head, int k) {
    //1. test weather we have more then k node left, if less then k node left we just return head 
    ListNode node = head;
    int count = 0;
    while (count < k) { 
        if(node == null)return head;
        node = node.next;
        count++;
    }

    // 2.reverse k node at current level 
    ListNode prev = reverseKGroup(node, k); //pre node point to the the answer of sub-problem 
    while (count > 0) {  
        ListNode cur = head;
        head = head.next;
        cur.next = prev;
        prev = cur;
        count = count - 1;
    }
    return prev;
}
```

# 复杂度
- 时间：`O(n)`
- 空间：`O(1)`
